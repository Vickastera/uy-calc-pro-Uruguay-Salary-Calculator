# 💰 UY Calc Pro

Una calculadora de salario para Uruguay que permite estimar sueldo **neto**, descuentos de **IRPF**, **FONASA** y escenarios de **renuncia o despido**, con visualización moderna tipo app fintech.

👉 Proyecto desarrollado con HTML, CSS y JavaScript puro.

---

## 🚀 Demo en vivo

🔗 https://vickastera.github.io/uy-calc-pro-Uruguay-Salary-Calculator/

---

## ⚙️ Funcionalidades

- 💰 Cálculo de sueldo bruto a neto
- 📊 IRPF según franjas
- 🏥 FONASA automático
- 🚪 Simulación de renuncia y despido
- 👨‍👩‍👧 Opción con hijos
- 📈 Gráficos de distribución (Chart.js)
- 📄 Exportación a PDF tipo recibo
- 📱 Diseño responsive tipo app fintech

---

## 🧠 Concepto

Este proyecto busca simular una **herramienta real de liquidación salarial en Uruguay**, útil para entender cuánto se descuenta realmente del sueldo y cómo impactan distintos escenarios laborales.

---

## 🛠️ Tecnologías usadas

- Chart.js
